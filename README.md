# socket-io-ile-chat-app
Socket io kullanarak chat uygulaması yapma

# Çalıştırmak için
## BACKEND
- cd backend
- yarn install
- yarn run dev

localhost:3000 adresi backend için kullanılacaktır.
## FRONDEND
- cd cliend
- yarn install
- yarn start

localhost:3001 adresi kullanıcı arayüzüdür. Birden fazla açarak deneme yapabilirsiniz.
